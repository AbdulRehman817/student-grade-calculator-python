# 🎓 Student Grade Calculator (Python)

A simple Python command-line project that calculates a student's total marks, average, grade, and performance remarks based on input marks of multiple subjects.

---

## 🚀 Features

- Takes input for multiple subjects
- Validates marks (0 to 100 only)
- Calculates total marks
- Calculates average percentage
- Assigns grades based on performance
- Gives performance remarks
- Handles invalid inputs gracefully

---

## 📊 Grading System

| Average Marks | Grade | Remark            |
|--------------|------|------------------|
| 90 - 100     | A+   | Outstanding      |
| 80 - 89      | A    | Excellent        |
| 70 - 79      | B    | Good             |
| 60 - 69      | C    | Average          |
| 50 - 59      | D    | Needs Improvement|
| Below 50     | F    | Fail             |
